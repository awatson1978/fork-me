# Fork Me

A smart package for showing those fucking github ribbons

## Usage

Basic

    new ForkMe({
      user: 'possibilities',
      repo: 'atmosphere'
    });

Custom

    new ForkMe({
      user: 'possibilities',
      repo: 'atmosphere',
      ribbon: {
        color: 'darkblue',
        position: 'left'
      },
    });
